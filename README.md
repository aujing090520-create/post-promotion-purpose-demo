# HelloTalk 帖文加热推广目的 Demo

用于评审帖文加热新增推广目的后的购买侧配置、受众侧互动效果和结果数据展示。

## 推广目的

- 默认推广
- 更多点赞：操作栏爱心动效引导
- 更多关注：关注按钮使用加热渐变
- 更多聊天：关注按钮替换为聊天入口，用户自行输入并发送消息

三个增强目的每次购买增加 30 COINS。

## 本地运行

```bash
npm install
npm run dev
```

## 验证

```bash
npm run build
```
